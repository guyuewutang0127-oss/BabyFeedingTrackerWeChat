# 宝宝喂养记录（微信小程序）

> 0-36 月龄婴幼儿喂养记录工具 · 本地存储 · 个人测试项目

## 项目简介

这是一个面向新手父母的婴幼儿喂养记录微信小程序，覆盖 0-36 月龄的喂养管理需求。项目使用微信小程序原生框架开发，数据全部存储在客户端本地。

**当前阶段：功能已实现，处于个人测试阶段**（未上架微信小程序平台）。

---

## 文件结构详解

### 小程序入口与全局配置

| 文件 | 类型 | 作用 |
|------|------|------|
| `app.js` | JavaScript | 小程序入口逻辑。初始化全局数据（宝宝信息、喂养记录、设置），加载喂养计划，检查是否需要喂养提醒 |
| `app.json` | JSON | 小程序全局配置。注册所有页面路由、配置 TabBar（底部导航栏：记录/历史/统计/设置）、设置导航栏样式（粉色主题） |
| `app.wxss` | WXSS | 全局样式。定义页面背景色、通用组件样式等 |
| `sitemap.json` | JSON | 微信搜索配置。允许小程序页面被微信索引 |

### 页面目录（pages/）

每个页面由 4 个文件组成：`.js`（逻辑）、`.wxml`（模板）、`.wxss`（样式）、`.json`（页面配置）

| 页面 | 路径 | 功能说明 |
|------|------|----------|
| **首页** | `pages/index/` | 核心操作页面。支持两种录入方式：①计时录入（开始/暂停/结束计时）②文字解析录入（如"下午3点喂了120ml"）。显示今日统计、上次喂养信息、下次喂养倒计时、科学喂养计划建议 |
| **历史页** | `pages/history/` | 查看所有喂养记录。支持左滑删除单条记录、按日期筛选、手动添加记录 |
| **统计页** | `pages/statistics/` | 数据可视化。展示今日/本周/本月的喂养次数、总奶量、平均奶量趋势图表 |
| **设置页** | `pages/settings/` | 配置中心。设置宝宝昵称/出生日期/性别/喂养方式、提醒间隔、科学喂养计划开关、灵活时间表模板选择、智能解析服务地址、数据备份/恢复/CSV导出 |
| **评估报告** | `pages/assessment/` | 展示 WHO 生长发育评估结果。根据已录入的身高体重数据，显示在同龄同性别儿童中的百分位水平 |
| **评估录入** | `pages/assessmentForm/` | 录入身高、体重、头围等数据，进行 WHO 标准评估 |
| **时间表** | `pages/feedingSchedule/` | 灵活时间表管理。可查看/调整系统推荐的每日喂养时间安排 |
| **手动添加** | `pages/addRecord/` | 手动录入单条喂养记录（补充计时和文字解析之外的录入方式） |

### 工具模块（utils/）

| 文件 | 作用 |
|------|------|
| `storage.js` | **本地存储封装**。封装 `wx.setStorageSync` / `wx.getStorageSync`，提供喂养记录的增删改查、数据格式化、备份恢复等功能。所有数据持久化都通过此模块 |
| `feedingPlan.js` | **科学喂养计划核心**。根据宝宝月龄自动匹配卫健委喂养指南，提供喂养间隔建议、奶量建议、辅食添加提示、灵活时间表模板（5种：新生儿/1-3月/3-6月/6-12月/12月+）。支持按性别和生长发育状态微调建议 |
| `parser.js` | **文字解析引擎**。支持两种模式：①规则解析（正则匹配口语化描述，如"下午3点泡了120ml喝了100ml"）②可选的 AI 解析（调用 parse-server 上的 Ollama 模型）。解析失败时自动降级到规则解析 |
| `charts.js` | **图表工具**。封装微信小程序 Canvas 绘图，生成喂养趋势折线图/柱状图，支持日/周/月视图切换 |
| `growthTracker.js` | **WHO 生长发育评估**。内置 WHO 0-5 岁儿童身高/体重/头围百分位标准数据，根据录入数据计算百分位并给出评价（偏瘦/正常/偏胖等） |
| `reminder.js` | **提醒逻辑**。计算距离上次喂养的时间，判断是否超过设定间隔，生成提醒文案 |

### 可选服务（parse-server/）

| 文件 | 作用 |
|------|------|
| `server.js` | Node.js 本地服务。调用 Ollama 本地小模型（如 qwen2:0.5b）理解口语化喂养描述，返回结构化 JSON。默认监听 0.0.0.0:3842 |
| `README.txt` | parse-server 的使用说明。包含 Ollama 安装、模型拉取、服务启动、小程序配置的详细步骤 |

### 静态资源（images/）

| 文件 | 用途 |
|------|------|
| `icon_*.png` / `icon_*_active.png` | TabBar 底部导航栏图标（普通状态/选中状态） |
| `default_baby.png` | 默认宝宝头像 |
| `empty_feeding.png` / `empty_history.png` | 空状态插图（无记录时显示） |
| `README.md` | 图标规格说明（尺寸、格式要求） |

| `project.config.example.json` | 配置模板。复制为 `project.config.json` 后填入你的 AppID |

## 技术栈

- **框架**：微信小程序原生（WXML / WXSS / JavaScript）
- **存储**：`wx.setStorageSync`（客户端本地存储）
- **图表**：微信小程序 Canvas API
- **可选 AI**：Node.js + Ollama 本地模型（实验性）

## 快速开始

### 1. 导入项目

```bash
cd baby-feeding-tracker

# 复制配置模板并填入你的小程序 AppID
copy project.config.example.json project.config.json
# 编辑 project.config.json，将 appid 改为你的 AppID

# 用微信开发者工具导入本目录，编译预览
```

### 2. 首次使用

1. 进入**设置页**，填写宝宝昵称、出生日期、性别、喂养方式
2. 返回**首页**，点击"开始计时"记录喂养，或切换到"文字录入"输入口语化描述
3. 在**历史页**查看记录，左滑可删除
4. 在**统计页**查看喂养趋势

### 3. 可选：启用智能解析

如需使用 AI 理解口语化描述：
1. 安装 [Ollama](https://ollama.com) 并拉取模型：`ollama pull qwen2:0.5b`
2. 启动 Ollama 后运行：`node parse-server/server.js`
3. 在手机与电脑同一局域网下，于设置页填写 `http://电脑IP:3842`
