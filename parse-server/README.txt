本地喂养文字解析服务（小模型）

作用：记录页「文字提取」在填写了解析服务地址后，会优先调用本服务，用本地小模型理解口语化表达（如“刚喂完”“喝了大概一百”），再回填到结构化结果。不配置或服务不可用时自动退回规则解析。

运行前准备
1. 安装 Ollama：https://ollama.com
2. 拉取小模型（任选其一，模型会下载到 Ollama 默认目录；若希望用 E:\models，可把 Ollama 的 OLLAMA_MODELS 环境变量设为 E:\models 后拉取）：
   ollama run qwen2:0.5b
   或 ollama run qwen2:1.5b
3. 确认 Ollama 已启动（本机一般会在 127.0.0.1:11434）

启动本服务
  node server.js
默认监听 0.0.0.0:3842。环境变量可选：
  PARSE_PORT=3842     （服务端口，默认 3842）
  OLLAMA_HOST=http://127.0.0.1:11434
  OLLAMA_MODEL=qwen2:0.5b

小程序配置
1. 手机与运行本服务的电脑需在同一局域网。
2. 在「设置」-「智能解析」中填写：http://电脑IP:3842（例如 http://192.168.1.100:3842）。
3. 模拟器调试时可直接填 http://127.0.0.1:3842（并勾选开发者工具「不校验合法域名」）。

模型未下载时请自行用 ollama pull 拉取；权重可放在 E:\models 并通过 OLLAMA_MODELS 指定。
